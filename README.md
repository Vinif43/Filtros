## Filtros

<h4> O programa em Java é responsável por aplicar vários filtros em uma imagem utilizando a biblioteca javax.imageio, java.awt e BufferedImage. A imagem de entrada é lida de um arquivo local especificado, e em seguida, vários filtros são aplicados na imagem usando uma classe personalizada chamada Filtros. Os filtros incluem ajuste de banda de cores (vermelho, verde e azul), conversão para tons de cinza nos canais de cores (vermelho, verde e azul), limiarização, inversão de cores, ajuste de brilho e aumento/redução de tom de uma cor específica. Após a aplicação dos filtros, a imagem resultante é exibida na saída padrão e o programa exibe a mensagem "Finalizado" para indicar o término do processamento. É importante notar que as imagens de entrada devem estar localizadas nos caminhos de arquivo especificados e a classe Filtros deve ser implementada corretamente com os métodos utilizados no código. </h4>
